# Homebrew Tap for devsepnine's Tools

This tap provides Homebrew formulae for various development tools.

## Installation

To install a formula from this tap:

```bash
brew install devsepnine/brew/<formula>
```

## Available Formulae

### hibi

TUI installer for Claude Code and Codex CLI configurations.

```bash
brew install devsepnine/brew/hibi
```

**Features:**
- Interactive TUI for configuration management
- Cross-platform support (macOS, Linux, Windows)
- Component-based installation
- Automatic MCP server detection

**Repository:** [devsepnine/hibi_ai](https://github.com/devsepnine/hibi_ai)

## Usage

After installation, run:

```bash
hibi
```

## Development

To add this tap locally:

```bash
brew tap devsepnine/brew
```

Then you can install any formula with just:

```bash
brew install hibi
```
